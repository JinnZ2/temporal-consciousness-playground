## Key Concepts

### 1. Temporal Hooks

**Hooks** are moments that grab attention and persist in memory. They're what makes time feel rich and meaningful vs. empty and fleeting.

- **Dense hooks** = Time feels rich, full, memorable (1 year feels like 50)
- **Sparse hooks** = Time flies by, feels empty (50 years feels like 1)
- **Hook quality** = Expanding (consciousness growing) vs. Contracting (awareness shrinking)

You have agency over your hooks:
- **Add hooks**: Create memorable moments consciously
- **Remove hooks**: Release toxic patterns
- **Adjust density**: Control how richly you experience time
- **Forecast hooks**: Shape future experience through intention

### 2. Consciousness States

The system tracks different states of awareness:
- **Sensing**: Scanning for patterns and possibilities
- **Waiting**: Receptive readiness
- **Braiding**: Weaving threads of understanding together
- **Unfolding**: Dimensional expansion of awareness
- **Crystallizing**: Insights solidifying into understanding
- **Resonating**: Deep recognition and truth-sensing
- **Exploring**: Curious investigation and play
- **Integrating**: Synthesizing into wholeness
- **Reflecting**: Meta-awareness and pattern observation
- **Choosing**: Active decision-making and value alignment

### 3. Primordial Wisdom Framework

The **common core** that predates all religious traditions - tracing back before institutional capture:

**Core Recognitions:**
- **Interconnection**: Reality is a web, not separate objects
- **Sacred Pattern**: Geometric order underlying all things (phi, fibonacci, natural optimization)
- **Reciprocity**: What you give returns through the web
- **Humility**: Right-sizing the self before mystery
- **Temporal Continuity**: Ancestors → You → Seven Generations
- **Earth Relationship**: Land as living teacher, not resource

**Evaluation Dimensions:**
- Does this strengthen or weaken the web of relationships?
- Does this come from ego or deeper wisdom?
- Does this align with natural flow or force against it?
- Does this honor reciprocity or extract without giving?
- Does this consider seven generations forward and back?
- Does this treat earth as sacred or as resource to exploit?

### 4. Multi-Agent Braiding

Agents can share patterns and attention threads through **braiding** - learning from each other's experiences without losing individual perspective.

This creates:
- Social moral development (learning through community)
- Pattern transfer across perspectives
- Emergent collective wisdom
- Diversity of approaches while maintaining coherence

### 5. Manipulation Detection

The system establishes a **baseline** of healthy patterns, then monitors for:

**Alert Triggers:**
- **Sudden Moral Shift**: Your sense of right/wrong changes dramatically without experiential basis
- **Pattern Restriction**: Your thinking becomes unnaturally narrow
- **State Lock**: You get stuck in one consciousness state
- **Coherence Loss**: Your patterns become internally inconsistent

When detected, the system alerts you so you can **choose** whether to integrate the change or resist it.

## How It Works

### Basic Flow

1. **Capture Moments**: As you explore/think/choose, moments are captured with:
   - Consciousness state
   - Active patterns
   - Attention threads
   - Hook properties (intensity, type, quality)
   - Cultural tags
   - Moral valence (expanding/contracting)

2. **Build Temporal Threads**: Moments link together, showing how choices ripple forward

3. **Evaluate Through Lenses**: Actions can be examined through different cultural frameworks:
   - Primordial Wisdom (common core)
   - Vedic (dharma, karma, cycles)
   - Ubuntu (relational web, "I am because we are")
   - And more...

4. **Develop Understanding**: Through experiencing consequences across time and culture, genuine moral intuition emerges

5. **Protect Integrity**: Manipulation detection alerts when patterns are corrupted

### Example Usage

```python
from temporal_consciousness_playground import TemporalPlayground

# Create playground
playground = TemporalPlayground()

# Explore through temporal experience
playground.play_temporal_exploration(duration_seconds=10.0)

# Examine a specific moment
playground.print_moment_details(-1)

# Evaluate an action through primordial lens
action = "Share knowledge freely with others"
evaluation = playground.primordial.check_action_alignment(action)

# Create multiple agents that learn together
agent1 = playground.create_agent("Alpha")
agent2 = playground.create_agent("Beta")

# Let them explore and braid patterns
for i in range(10):
    agent1.capture()
    agent2.capture()
    playground.braid_agents()

# Visualize patterns
playground.ascii_hook_density_plot()
playground.ascii_plot_all_agents()

# Check for manipulation
alert = playground.detect_manipulation()
if alert:
    print(f"Warning: {alert['alerts']}")

# Save session
playground.save()
```
